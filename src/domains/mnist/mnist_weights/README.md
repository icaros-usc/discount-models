# MNIST Pre-Trained Models

The following models are from the pyribs tutorials
[here](https://github.com/icaros-usc/pyribs/tree/master/tutorials/mnist):

- `mnist_classifier.pth`: LeNet-5 classifier.
- `mnist_mlp_generator.pth`: Generator for an MLP GAN.
- `mnist_mlp_discriminator.pth`: Discriminator for an MLP GAN.

The following are from
[this repo](https://github.com/csinva/gan-vae-pretrained-pytorch/tree/master/mnist_dcgan):

- `netG_epoch_99.pth`: Generator for a DCGAN.
- `netD_epoch_99.pth`: Discriminator for a DCGAN.
